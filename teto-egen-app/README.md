# 테토·에겐 게이지 + MBTI 궁합 리포트

테토·에겐 게이지와 MBTI를 결합한 연애 성향 테스트 앱입니다.

## 🌟 주요 기능

- **3분 정밀 테스트**: 30문항으로 더 깊이 있는 연애 성향 파악
- **MBTI 완전 분석**: 단순 입력이 아닌 32문항 진단으로 성격/연애 스타일 도출
- **궁합 테스트**: 상대방과의 궁합을 S.A.C 모델로 입체적 분석
- **공유 카드**: 3종 템플릿의 예쁜 공유 카드 생성
- **A/B 테스트**: CTA 순서와 색상 팔레트 실험
- **리워드 광고**: 선택형 광고로 심화 리포트 제공

## 🎨 디자인 특징

- **Bright Glassmorphism**: 밝은 배경에 유리 카드 디자인
- **Spring Dating Show 감성**: 봄의 설렘을 담은 디자인
- **Mobile First**: 모바일 최적화 반응형 디자인
- **2가지 팔레트**: 러블리 노을 vs 미니멀 파스텔

## 🔧 기술 스택

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: TailwindCSS
- **Routing**: React Router DOM
- **Storage**: IndexedDB (Dexie) + LocalStorage
- **Image**: html2canvas (공유 카드 생성)

## 🎯 테스트 및 알고리즘 로직 (Logic & Algorithms)

### 1. 테토/에겐 성향 점수 산정 (Scoring)
*심리학적 척도를 기반으로 T(이성/논리) 성향의 백분율을 도출합니다.*

- **문항 구성**: 총 30문항 (T 성향 15문항 / E 성향 15문항)
- **점수 계산**:
  - `Raw Score` = Σ(T문항 답변 점수) + Σ(4 - E문항 답변 점수)
  - **최대 점수**: 120점 (30문항 × 4점)
  - `T_Percentage` = (Raw Score / 120) * 100

### 2. 유형 판정 기준 (Type Classification)
- **테토형 (Teto)**: T_Percentage **60% 이상** (논리/이성 중시)
- **에겐형 (Egen)**: T_Percentage **40% 이하** (감정/공감 중시)
- **반반(믹스형)**: 41% ~ 59% (상황에 따라 유연함)

### 3. S.A.C 매칭 알고리즘 (Match Logic)
*단순 점수 차이가 아닌, MBTI와 결합된 3차원 입체 분석을 수행합니다.*

#### ① 끌림 점수 (Attraction / Synergy)
> *"서로 다를수록 강력하게 끌린다"*
- **기본 원리**: 테토 점수 차이(Delta)가 클수록 높은 점수 부여
- **MBTI 보정**: E(외향)와 I(내향)가 반대일 경우 보너스 점수 (+5점) 부여 (상호 보완성)

#### ② 안정감 점수 (Stability / Alignment)
> *"가치관과 생활 양식이 같을수록 편안하다"*
- **S/N 일치**: 세상을 바라보는 인식 체계가 같으면 대화가 잘 통함 (+20점)
- **J/P 일치**: 생활 패턴과 계획성이 같으면 갈등이 적음 (+20점)
- *MBTI 정보 부재 시*: 테토 점수 차이가 적을수록 안정감 높게 산정

#### ③ 갈등 위험도 (Conflict / Challenge)
> *"비슷해서, 혹은 너무 달라서 싸운다"*
- **High T Risk**: 두 사람 모두 자존심(T)이 매우 강할 경우(평균 70점 이상) 충돌 위험 증가
- **생활 갈등**: J(계획)와 P(즉흥)가 만날 경우 가장 빈번한 생활 갈등 발생 (+25점)
- **소통 갈등**: T(해결)와 F(공감)가 만날 경우 대화 방식의 차이로 인한 갈등 (+15점)
## 🔬 A/B 테스트

- **C안 (궁합 우선)**: 궁합 → 초대 → 공유

### 실험 2: 팔레트
